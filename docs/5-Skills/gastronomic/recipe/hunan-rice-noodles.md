# 湖南米粉

## 食材

1.   五花肉一斤（半肥半瘦）
2.   米粉
3.   姜
4.   小葱
5.   老抽
6.   生抽

## 步骤

1.   **切肉**：皮、肥、瘦分开
2.   **冷水下锅煮开**：瘦肉 + 姜片。把血水煮出来。
3.   **洗肉**
4.   **原汤肉丝码**：1.5L水 + 葱头 + 姜片 + 一点生抽 + 一点老抽 + 瘦肉。大火煮开后，关最小的火煲两个小时，将肉煮烂。
5.   **熬煮猪油**：小火煎，铲子压。
6.   **做碗**：小葱 + 一点老抽 + 盐 + 两勺猪油
7.   **煮粉**：开水下锅煮粉，沸腾后加一次冷水，煮开捞出来，加两勺原汤。
