**自动化**

-   输入一对HTML标签：输入标签名，再按Tab键
-   生成 getter 和 setter：右键 generate

**缩进**

-   缩进：Tab
-   反向缩进：Shift+Tab

 **注释**

-   行注释/取消行注释：Ctrl+/
-   **块注释/取消块注释：Ctrl+Shift+/**

**输入** 

-   **插入多个光标：Alt+Shift+点击**

-   **复制当前行：Ctrl+D**
-   **删除当前行：Ctrl+Y**
-   **移动当前行：Alt+Shift+上下方向键**
-   **合并行：Ctrl+Shift+J**

 **文件**

-   **优化导入：Ctrl+Alt+O  即：删除无用的import**

 **外观**

-   动态改变字体的大小：Ctrl+鼠标滚动

 **调试**

-   打开浏览器的开发者工具：F12
-   浏览器的刷新：Ctrl+R或者F5
