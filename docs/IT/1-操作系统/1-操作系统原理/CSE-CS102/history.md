# History

## 1960s

-   1964, BASIC

## 1970s

-   1969, 1971, 1973 - Unix
-   1970 - Pascal (programming language)

-   **1972 - C programming language**

-   1976 - Apple I

-   1977 - Apple II series

-   **1978 - K&R C, *The C Programming Language* 1st Edition**
    -   1988 - *The C Programming Language* 2nd Edition

## 1980s

-   **1989, 1990 - ANSI C, ISO C**
-   **1999 - C99**
