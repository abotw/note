    Headings, Headers 标题¶

    Markdown: `#` hash tag/symbol
    # Heading 1
## Heading 2
### Heading 3
#### Heading 4
##### Heading 5
###### Heading 6

OR

Heading 1
=========
Heading 2
---------
    HTML
    <h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

    Markdown
    This is the first paragraph.

This is the second paragraph of text.

    This is the first paragraph.

    This is the second paragraph of text.

    HTML
    <p>This is a paragraph.</p>

    This is a paragraph.

    Code Blocks 代码块¶

    Dots indicate leading spaces.

    Markdown
    ```
This is a fenced code block.
# code block
print '3 backticks or'
print 'indent 4 spaces'
```

····# code block
····print '3 backticks or'
····print 'indent 4 spaces'
    HTML
    <pre>
  <code>
  function test() {
    console.log("html");
  }
  </code>
</pre>

    Tables Are Cool
    col 1 is left-aligned $100
    col 2 is centered $52
    col 3 is right-aligned $9
    Markdown
     Header 1 | Header 2 
----------|----------
 Cell 11  | Cell 12  
 Cell 21  | Cell 22  

|  Tables  |      Are      | Cool |
|----------|:-------------:|-----:|
| col 1 is |  left-aligned | $100 |
| col 2 is |    centered   |  $52 |
| col 3 is | right-aligned |   $9 |

    <table>
  <thead>
    <tr>
      <th>Header 1</th>
      <th>Header 2</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Cell 11</td>
      <td>Cell 12</td>
    </tr>
    <tr>
      <td>Cell 21</td>
      <td>Cell 22</td>
    </tr>
  </tbody>
</table>

    Emphasis¶

    Italic 斜体

    A single set of asterisks.

    Markdown
    I am *italic*
I am _Italic_
    HTML
    I am <i>italic</i>

    Bold 粗体

    Double asterisks.

    Markdown
    I am **bold**
I am __bold__
    HTML
    I am <b>bold</b>

    Strikethrough

    Markdown
    I am ~~strikethrough~~
    HTML
    I am <del>strikethrough</del>

    Underlined

    This was inserted

    Markdown
    ^^This was inserted^^
    HTML
    <ins>This was inserted</ins>

    ==Text to be highlighted==
    HTML
    <mark>Text to be highlighted</mark>
